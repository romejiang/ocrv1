## 中国体育彩票概述

### 彩票类型

中国体育彩票足球彩票主要类型：
- **胜平负**: 比赛结果 3 选 1（主胜/平/客胜）
- **比分**: 预测具体比分
- **半全场**: 上半场+全场结果组合
- **进球数**: 预测总进球数

### 票面结构

典型足彩彩票票面包含：
1. **期号**: 如 "第 25042 期"
2. **投注时间**: 如 "2026-04-16 14:30"
3. **投注内容**: 比赛列表，每场比赛包含：
   - 比赛信息: 主队 vs 客队
   - 投注选项: 胜/平/负 或 具体比分
   - 赔率: 如 1.85
4. **投注倍数/金额**: 如 "66 元"
5. **多注**: 单张票可能包含多注投注

## 解析策略

### 期号提取 (LOT-01)

正则模式:
```python
issue_pattern = r'第\s*(\d{5,6})\s*期'
# 或
issue_pattern = r'期号[:：]?\s*(\d{5,6})'
```

匹配示例:
- "第 25042 期" → "25042"
- "25042期" → "25042"

### 时间提取 (LOT-02)

正则模式:
```python
time_pattern = r'(\d{4}[-/]\d{2}[-/]\d{2}[T\s]?\d{2}:\d{2}(?::\d{2})?)'
# 或简化版
date_pattern = r'(\d{4}[-/]\d{2}[-/]\d{2})'
```

### 金额提取 (LOT-03)

正则模式:
```python
amount_pattern = r'投注金额[:：]?\s*[￥$]?\s*(\d+(?:\.\d{1,2})?)'
# 匹配 "投注金额：￥66" 或 "66元" 或 "66.00"
```

### 投注内容解析 (LOT-04)

彩票投注内容结构:
```
阿根廷 vs 巴西
  胜:1.85  平:2.10  负:3.50
```

或者横排格式:
```
阿根廷 胜 1.85  巴西 负 3.50
```

解析策略:
1. 识别 "vs" 或 "VS" 分隔比赛双方
2. 在赔率附近查找选项（胜/平/负）
3. 赔率通常是 xx.xx 格式的小数

### 赔率提取 (LOT-05)

正则模式:
```python
odds_pattern = r'(\d+\.\d{2})'
# 匹配 1.85, 2.10, 3.50 等
```

### 多注识别 (LOT-06)

多注识别策略:
1. 赔率数值出现多次 → 多注
2. 检查 "注" 关键字: "第1注", "第2注"
3. 根据位置分行分隔

## 潜在问题

### 1. OCR 识别错误

OCR 可能将 "0" 识别为 "O"，"1" 识别为 "l"
解决: 后处理校验，如期号必须全为数字

### 2. 版面格式差异

不同彩票公司或不同年份格式可能不同
解决: 支持多种正则模式匹配

### 3. 赔率与选项对应

赔率和选项可能在不同行
解决: 基于位置推断对应关系

### 4. 混合语言

彩票可能有中英文混合
解决: 优先中文匹配，英文作为补充
