## Goal

实现彩票基础字段（期号、时间、金额）的正则表达式提取。

## Implementation

### LotteryParser 类

位置: `src/parser.py`

```python
class LotteryParser:
    ISSUE_PATTERN = re.compile(r"(?:第\s*)?(\d{5,6})\s*期")
    DATE_PATTERN = re.compile(r"(\d{4}[-/]\d{2}[-/]\d{2})")
    AMOUNT_PATTERN = re.compile(r"投注金额[:：]?\s*[￥$]?\s*(\d+(?:\.\d{1,2})?)")
```

### 正则表达式模式

| 字段 | 模式 | 匹配示例 |
|------|------|----------|
| 期号 | `(?:第\s*)?(\d{5,6})\s*期` | "第 25042 期" → "25042" |
| 时间 | `(\d{4}[-/]\d{2}[-/]\d{2})` | "2026-04-16" → "2026-04-16" |
| 金额 | `投注金额[:：]?\s*[￥$]?\s*(\d+(?:\.\d{1,2})?)` | "投注金额：￥66" → 66.0 |

## Verification

parser = LotteryParser()
